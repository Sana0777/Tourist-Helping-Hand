# Tourist Helping Hand

This is a Django-based web application created to help tourists explore
places such as hotels, temples, restaurants, shopping malls, and tourism locations.

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite
- Bootstrap 

## Project Structure
This project contains multiple Django apps like:
- Hotels
- Restaurants
- Temples
- TourismPlaces
- Accounts

Static files and images are currently scattered and need refactoring.

## How to Run (Basic)
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run server:
   python manage.py runserver
